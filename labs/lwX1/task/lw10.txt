1.  Разработайте механизм проверки надежности пароля пользователя для формы Sign up.
#25 При окончании заполнения поля New Password необходимо асинхронно 
    отправлять пароль пользователя серверному PHP скрипту при помощи AJAX запроса. 
    Скрипт должен дать оценку надежности пароля 
    по трехбальной шкале (0 - weak, 1 - good, 2 - strong). 
    Алгоритм оценки разработайте самостоятельно. 

    Справа от поля ввода пароля необходимо отображать результат проверки. 
    Красным слово weak, желтым - good и зеленым - strong.



2.  Разработайте механизм выставления рейтингов файлов пользователей. 
#30 В списке файлов необходимо добавить примерно такой блок 
    -1 234 +1, 

    где 234 текущий рейтинг файла. 


    При нажатии на -1 и +1 необходимо посылать AJAX запрос серверу для пересчета рейтинга.
    PHP обработчик должен возвращать новое значение рейтинга, которое надо показать в блоке.

    В БД необходимо сохранять какой пользователь и каким образом изменял рейтинг файла.
