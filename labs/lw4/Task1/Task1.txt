Требуется:
  Модифицировать приведенные шаблонные файлы index.html, index.css таким образом, 
    чтобы получилась страница, оформление которой аналогично приведенному скриншоту Task1.png
  Страница должна точь-в-точь совпадать с приведенным скриншотом (попиксельное соответствие).
  Должна быть реализована кросс-браузерность (оформление должно быть аналогичным во всех современных браузерах, включая Internet Explorer 7-9)

Запрещается:
  Использовать позиционирование (свойство position)
  Использовать хаки (для достижения кросс-браузерности)

Разрешается пользоваться готовыми стилями, имеющимися в файлах style.css, index.css. (left, right, clear, ...)
Для выполнения задания в разметке достаточно использовать тег <div>.
Имена классов для однотипных элементов разрешается задавать в виде layer1, layer2, layer3 и т.д.
